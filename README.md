## Overview

Small translators for Brazilian indigenous languages, made while learning Python and computational linguistics

Created to help preserve endangered native languages

## Changelog

### [v0.1.0] - 2026-04

* Initial release
* Word-based Yaathe-to-Portuguese translator
* Sentence-based Tupi Potiguara-to-Portuguese-and-English translator
* Limited vocabulary and features for now
